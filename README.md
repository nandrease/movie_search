# movie_search
Front-End React, Backend NestJS

## Prereqs
- Node.js (LTS recommended)

## Structure
- `backend/`: NestJS API
- `frontend/`: React (Vite) app

## Install

```bash
cd backend
npm install

cd ../frontend
npm install
```

## Run (dev)

### Backend (NestJS)

```bash
cd backend
npm run start:dev
```

Default: `http://localhost:3000`

### Frontend (React)

```bash
cd frontend
npm run dev
```

Vite will print the local URL (typically `http://localhost:5173`).
